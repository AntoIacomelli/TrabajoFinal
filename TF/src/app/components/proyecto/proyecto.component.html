Portfolio

Este protecto se creó como trabajo final para Argentina Programa, en el cual debiamos plasmar todo lo que habíamos aprendido durante el cuerso de programación. Este trabajo final consiste en crear un portfolio personal con nuestros datos como educación, experiencias laborales, proyectos, Hard&Soft skills etc. El portfolio debía contar con un Frontend, donde se debia usar HTML, CSS, JAVA etc y con un Backend, donde se armó una base de datos con el uso de MySQL, NetBeans etc.
